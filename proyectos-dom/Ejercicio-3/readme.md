## Ejercicio: Añadir y quitar estilos CSS con eventos

### Objetivo
El objetivo de este ejercicio es aprender a modificar estilos CSS dinámicamente desde JavaScript utilizando eventos del usuario.

En concreto:
- Detectar cuándo el ratón entra y sale de una imagen
- Aplicar estilos directamente desde JavaScript
- Entender cómo funciona la propiedad `style` en el DOM

---

## Comportamiento del ejercicio

- Cuando el usuario pasa el ratón sobre la imagen:
  - Se añade un borde rojo
  - La imagen se vuelve circular

- Cuando el ratón sale de la imagen:
  - El borde desaparece (se vuelve transparente)
  - Se mantiene el estado original visual
