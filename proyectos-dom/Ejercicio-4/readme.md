## Ejercicio: Mostrar y ocultar elementos con `classList.toggle`

### Objetivo
El objetivo de este ejercicio es aprender a mostrar y ocultar elementos del DOM utilizando clases CSS dinámicas y eventos en JavaScript.

En concreto:
- Detectar clics sobre un botón
- Alternar clases CSS dinámicamente
- Comprender el funcionamiento de `classList.toggle()`



## Comportamiento del ejercicio

- El contenido empieza oculto mediante la clase `hidden`
- Cuando el usuario hace clic en el botón:
  - El contenido aparece
- Si vuelve a hacer clic:
  - El contenido se oculta nuevamente
